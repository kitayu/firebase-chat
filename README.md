# firebase-chat

## 概要
このアプリケーションは、書籍「Testable Firebase接続可能なFirebaseプロジェクトを目指して」の学習で作成した、
Firebaseを使ったウェブアプリケーションです。

## 実行手順
1. `yarn install`
2. `yarn dev`
